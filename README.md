# # Rover Hardware
Hardware design(Mechaical & Electrical design files) and documentation for the rover, including schematics, wiring diagrams, components, embedded system interfaces, CAD models(fusion 360).



This repository contains all mechanical and electrical designs for the Mars Rover project.

## Contents
- Mechanical design (CAD, chassis, structure)
- Electrical systems (schematics, wiring, PCB)
- Components and parts list (BOM)
- Hardware images and prototypes

## Structure
- mechanical/
- electrical/
- components/
- images/
